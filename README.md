# 🧮 Python Menu-Driven Calculator

A simple console-based calculator built with Python.  
This project demonstrates the use of **loops, conditionals, and user input handling** to perform basic arithmetic operations.

---

## ✨ Features
- Addition, subtraction, multiplication, and division
- Menu-driven interface for easy navigation
- Error handling for division by zero
- Supports floating-point numbers
- Runs continuously until the user chooses to exit

---

## 🚀 How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com//mukeshrg2002-hash/menu-calculator.git
   cd menu-calculator
   python calculator.py
   ```
